Política de Privacidade - SEI Helper

Coleta de Dados

Esta extensão NÃO coleta, armazena ou transmite nenhum dado pessoal do usuário.

Dados Processados

A extensão opera apenas localmente no navegador do usuário e:

  • Modifica a interface visual do sistema SEI para melhorar a experiência do usuário
  • Substitui números de processo por assuntos correspondentes
  • Substitui CPFs por nomes dos responsáveis
  • Aplica destaque visual em processos com marcadores
  • Armazena localmente apenas a preferência de animação (ON/OFF)

Armazenamento

Todos os dados processados ficam restritos ao navegador do usuário e não são transmitidos para servidores externos.

Permissões

As permissões solicitadas são utilizadas exclusivamente para:

  • storage: Salvar a preferência de animações localmente
  • Acesso ao SEI: Modificar a interface do sistema
  • activeTab: Aplicar estilos e manipular conteúdo da página
